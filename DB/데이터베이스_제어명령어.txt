SQL 기초 명령어 정리

데이터 정의어 DDL
	
데이터를 생성,변경,제거

CREATE
ALTER
DROP
RENAME
TRUNCATE
COMMENT


데이터 조작어 DML

테이블에 새 데이터를 삽입하거나, 테이블에 저장된 데이터를 수정 삭제 검색하는 기능을 제공

SELECT
INSERT
UPDATE
DELETE
MERGE



데이터 제어어 DCL


보안을 위해 데이터에 대한 접근 및 사용 권한을 사용자별로 부여하거나 취소하는 기능을 제공

GRANT
REVOKE


트랜잭션 제어어 TCL

데이터베이스의 논리적 단위작업인 트랜잭션 단위로 작업내용을 저장 또는 취소하는 기능을 제공

COMMIT
ROLLBACK
SAVEPOINT
